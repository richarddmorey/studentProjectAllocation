Options

Click the menu icon above (☰) to access the app options.

Option Default Description
Randomize before FALSE Randomize the student order at the start? Students higher up in the list will get first consideration. This may be desired. If not, then you might randomize the students to increase fairness.
Distribute unallocated TRUE Select this option to randomly distribute the unallocated students after the algorithm runs. If this option is not selected, then unallocated students will remain unallocated.
Output delimiter ; Character used to paste together multiple elements in a field in the lecturer and project output files.
Time limit 15 Limit (in seconds) on the time the algorithm runs
Iteration limit 0 Limit on the number of iterations for the algorithm (0 for no limit)
